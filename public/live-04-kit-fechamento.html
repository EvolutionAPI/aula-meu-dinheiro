Autoavaliação

Rubrica — como você se saiu?

Avalie cada item de 0 a 2. Mínimo para aprovação: 8 de 10.

1. Escopo claro (0/1/2)

  • 0 — Não definiu problema ou escopo ficou genérico
  • 1 — Problema definido mas escopo transbordou ou ficou vago
  • 2 — Problema em 1 frase, MVP fechado, out-of-scope explícito

2. Execução guiada por artefatos (0/1/2)

  • 0 — Pulou etapas ou gerou docs sem usar
  • 1 — Gerou docs mas não seguiu na implementação
  • 2 — Pipeline EVO seguida — cada doc alimentou o próximo

3. Deploy funcional (0/1/2)

  • 0 — Não fez deploy ou app não funciona
  • 1 — Deploy feito mas com erros ou sem domínio
  • 2 — App em produção, domínio com HTTPS, fluxo completo funcionando

4. Qualidade mínima (0/1/2)

  • 0 — Sem review, commits bagunçados
  • 1 — Review parcial ou commits sem padrão
  • 2 — Code review antes de cada commit, commits semânticos organizados

5. Fluência no workflow EVO + Claude (0/1/2)

  • 0 — Não conseguiu operar os comandos sozinho
  • 1 — Operou com ajuda constante, travou em vários momentos
  • 2 — Navegou o workflow com autonomia, soube destravar quando travou

Próximos 7 dias

Plano de consolidação

Micro-tarefas para fixar o que foi aprendido — sem guia, sem live, por conta própria.

  1. Dias 1–2: Adicione 1 feature pequena sozinho
    Escolha algo simples (filtro por categoria, edição de transação, tema dark/light). Use o workflow EVO: crie a story, implemente, review, commit, push.
  2. Dia 3: Faça deploy da mudança
    Push na main, acompanhe o build na Vercel, valide em produção. Se quebrar, use o troubleshooting do guia da live para resolver.
  3. Dias 4–5: Reforce o que ficou fraco
    Se a rubrica mostrou nota baixa em algum item, dedique esse tempo para reforçar. Releia o artefato correspondente e refaça se necessário.
  4. Dias 6–7: Documente o que aprendeu
    Escreva 1 parágrafo respondendo: “O que eu sei fazer agora que não sabia antes da Classe 1?”
O objetivo não é fazer mais features — é provar para si mesmo que consegue operar o ciclo completo sem ninguém segurando sua mão.
